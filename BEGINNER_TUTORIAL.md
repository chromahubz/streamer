## â“ Common Questions

### "Why isn't chat appearing?"

**Check:**
1. Did you click "Start Capture" for chat monitor?
2. Is the correct monitor selected? (Should show chat overlay)
3. Is chat visible on that screen? (Not minimized/hidden)
4. Look at top of app â†’ Does it say "Chat Capturing" badge?

**Fix:**
- Click "Get Available Sources" again
- Make sure chat is clearly visible on the selected screen
- Try increasing the monitor size (make chat bigger/clearer)

### "Responses are slow or not generating?"

**Check:**
1. Top of app â†’ Is "Backend: Connected" badge green?
2. Did you wait 3-5 seconds? (AI needs time to think!)
3. Is your internet working?

**Fix:**
- Check that backend is running (should see green "Connected")
- Refresh the app (close and reopen)
- Run `node test-apis.js` to test APIs

### "TTS isn't playing?"

**Check:**
1. Did you select a response first? (Click on one)
2. Is your volume turned up?
3. Is Groq API working? (Check API_STATUS.txt)

**Fix:**
- Make sure you clicked ON a response (it should highlight)
- Check browser console for errors (F12 key)
- Test with: `node test-apis.js`

### "Translation isn't working?"

**Check:**
1. Did you toggle "Enable Translation" to ON?
2. Did you select source and target languages?
3. Are they different languages? (Can't translate English â†’ English!)

**Fix:**
- Go to Setup tab â†’ Translation Settings
- Make sure toggle is blue (ON)
- Pick different source and target languages

### "App crashed or froze?"

**Fix:**
1. Close the app (Cmd+Q or close window)
2. In terminal, stop all processes (Ctrl+C)
3. Restart: `npm run dev`
4. Should relaunch fresh!
